# code
编码解码的便捷工具
修改，添加请在Main.py和Main_tools.py里面进行
团队：Thirteen Team
作者：crazy
使用方法：
Windows：python main.py -h
Linux：python main.py -h
工具支持：
url编码/解码,base64编码/解码,十六进制编码/解码,ASCII转码/解码,mad5加密/解密

options:
  -h, --help  show this help message and exit
  -u U        url编码
  -ul UL      url解码
  -s S        base64编码
  -sl SL      base64解码
  -x X        十六进制编码
  -xl XL      十六进制解码
  -a A        ascii编码
  -al AL      ascii解码
  -t T        md5加密--生成16位，32位
  -tl TL      md5解密--支持16位，32位
